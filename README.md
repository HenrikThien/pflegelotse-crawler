# Pflegelotse Crawler
Dieses Programm benutzt die Seite pflegelotse um Benutzersuchen zu automatisieren.

## Installation Voraussetzungen
<ul>
    <li>Python3.7+</li>
    <li>Dependencies installieren</li>
    <li>Chrome Browser</li>
    <li>Chromedriver</li>
</ul>

## Releases

Basis Version
